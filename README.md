# 🦠 Malaria Detection using Deep Learning

An AI-powered web application for detecting malaria from microscopic blood cell images using a CNN model and Flask backend.

---

## 🚀 Features

- Detects: **Parasitized / Uninfected**
- Confidence score visualization
- Dynamic diagnostic report page
- Modern glassmorphism UI
- Secure image validation
- Clean project structure

---

## 🛠 Tech Stack

- Python
- Flask
- TensorFlow / Keras
- NumPy
- Pillow
- HTML / CSS / JavaScript

---

## 📂 Project Structure

```
Malaria-Detection-using-Deep-Learning
│
├── app.py                # Flask application

├── .gitignore

    ├── index.html        # Upload interface
```

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```bash
git clone https://github.com/NANDANI2122/malaria-detection-dl.git
cd malaria-detection
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

Activate:

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Application

```bash
python app.py
```

Open browser:

```bash
http://127.0.0.1:5000
```
